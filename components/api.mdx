# API

A simple JSON API to events on this site.

## Endpoints

- [`/api/events/upcoming`](/api/events/upcoming) – all upcoming events
- [`/api/events/all-upcoming`](/api/events/upcoming) – all upcoming, grouped by month
- [`/api/events/all`](/api/events/all) – all events ever
- [`/api/events/all-monthly`](/api/events/all) – all events, grouped by month
- [`/api/amas`](/api/amas) – all AMAs ever

## Event Schema

- IDs are internal to our Airtable.
- Descriptions are in plain text but sometimes use Markdown formatting.
