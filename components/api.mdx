# API

A simple JSON API to events on this site.

## Filtering

You can filter events by tag via the `?tags=` query parameter. Multiple tags are comma-separated.

- [`/api/events/upcoming?tags=stardance`](/api/events/upcoming?tags=stardance)
- [`/api/events/all?tags=ama,workshop`](/api/events/all?tags=ama,workshop)

## Endpoints

- [`/api/events/upcoming`](/api/events/upcoming) – all upcoming events
- [`/api/events/upcoming-monthly`](/api/events/upcoming-monthly) – all upcoming, grouped by month
- [`/api/events/all`](/api/events/all) – all events ever
- [`/api/events/all-monthly`](/api/events/all-monthly) – all events, grouped by month
- [`/api/amas`](/api/amas) – all AMAs ever

## Event Schema

- IDs are UUIDs.
- Descriptions are formatted in markdown.
- `tags` is an array of strings (e.g. `["stardance"]`, `["ama", "workshop"]`).
